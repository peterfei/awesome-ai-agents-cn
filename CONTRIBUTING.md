# 贡献指南

## 提交案例

欢迎提交新的 AI Agent 落地案例。每条案例需按以下模板填写。

### 文件位置

`cases/<你的案例名>.md`

### 模板

```markdown
---
title: 案例标题
author: @你的GitHub用户名
date: 2026-MM
stack: 技术栈，逗号分隔
difficulty: 入门 / 进阶 / 专家
---

## 场景

描述这个 Agent 解决的问题。让读者一眼判断是否和自己相关。

## 实现

核心方案描述。包括：
- 技术选型理由
- 关键配置/代码片段
- 架构要点

## 效果

量化的效果数据优先：
- 效率提升百分比
- 人工介入减少量
- 准确率/覆盖率

定性的效果也可以：
- 团队反馈
- 上线后的变化

## 要点

实践中遇到的坑和总结的经验。

## 相关资源

- 链接1
- 链接2
```

### 提交流程

1. Fork 本仓库
2. 在 `cases/` 下创建你的案例文件
3. 提交 PR
4. 审核合并
